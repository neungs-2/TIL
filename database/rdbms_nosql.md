# **RDBMS vs NoSQL**

|             |                    RDBMS                    |                   NoSQL                   |
| :---------: | :-----------------------------------------: | :---------------------------------------: |
| 데이터 모델 |    정규화와 참조 무결성이 보장된 스키마     |    스키마가 없는 자유로운 데이터 모델     |
|  트랜젝션   |              강력한 ACID 지원               |            완화된 ACID (BASE)             |
|    확장     |           하드웨어 강화(Scale up)           | 수평 확장 가능한 분산 아키텍처(Scale out) |
| 적합한 용례 | 데이터 정합성이 보장되어야 하는 은행 시스템 | 낮은 지연시간, 가용성이 중요한 SNS 시스템 |
